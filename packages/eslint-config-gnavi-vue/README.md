# eslint-config-gnavi-vue

[ESLint](http://eslint.org/) config for Vue.js Project of gnavi

## Install

```sh
$ npm i --D eslint-config-gnavi-vue
```

## Usage

In your `.eslintrc.js`:

```
modules.export = {
  extends: ["gnavi-vue"]
}
```

or in your `package.json`:

```json
"eslintConfig": {
  "extends": ["gnavi-vue"]
}
```
